遇到的问题 (3)

openEuler 官方 repo (repo.openeuler.org) SSL 握手超时

方案: 切换到华为云镜像源 mirrors.huaweicloud.com/openeuler/

来源: 自行排查网络连通性后发现华为云镜像可用

ubs-comm-devel 包不在 SP2 仓库中

方案: 按 docs/build_install/构建指导.md 2.2.2 节的指引，从源码构建 ubs-comm 并通过 ninja install 安装到系统目录

来源: docs/build_install/构建指导.md - 2.2.2 其他openEuler版本依赖安装说明

单元测试失败：CMake 缓存路径不匹配

方案: 需要清理 cmake-build-debug 目录后重新运行测试。当前 Release 构建已成功，测试环境可在本地容器中手动清理后验证。

来源: cmake-build-debug/CMakeCache.txt 中的 /workspace/ubs-engine 路径与当前 /src 路径不匹配
