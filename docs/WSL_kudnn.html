文档缺失 (6)

#1HPCKit 下载链接不明确 - README 指向 hikunpeng.com，该页面为 JavaScript SPA，需要手动浏览器登录华为鲲鹏社区账号才能获取实际下载链接
#2无备选依赖源 - 文档未说明 HPCKit 各组件 (Bisheng, KML, KUPL) 是否有独立安装包或 openEuler RPM 仓库可用
#3架构限制未醒目提示 - README 在版本配套章节列出了 AArch64 架构要求，但未在概述/快速开始等显眼位置强调此限制
#4KML/KUPL 头文件和 API 文档缺失 - 源码中大量引用 kupl 和 kblas 库的头文件，但文档未提供这两个库的 API 参考链接
#5pip 镜像源未说明 - 卷积 kernel 代码生成需要 python3，但文档未说明是否需要额外 pip 包
#6googletest 备用下载方案不完整 - 文档提到网络问题时可手动下载 googletest，但仅提供 GitHub 链接

遇到的问题 (3)

HPCKit 无法自动化下载
QEMU 模拟 ARM 架构可能不完全支持 SVE/SVE2/SME 指令
HPCKit 下载页面需登录认证
