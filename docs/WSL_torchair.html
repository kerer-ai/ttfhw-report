遇到的问题 (4)

使用清华源安装 PyTorch 时，默认下载带 CUDA 的大版本（2-4GB），下载时间过长

方案: 改用 PyTorch CPU-only 版本 (https://download.pytorch.org/whl/cpu)，只需 192MB

来源: 用户反馈

缺少 python3-devel 导致 Python.h 头文件找不到

方案: 安装 python3-devel 包

来源: CMakeLists.txt 中 find_package(Python3) 需要开发头文件

缺少 wheel 包导致 Python wheel 打包失败

方案: 安装 wheel 包

来源: CMakeLists.txt 中使用 setup.py bdist_wheel 命令

torchair 运行时需要 CANN SDK (昇腾软件栈)，未在容器中安装

方案: 根据文档说明，CANN SDK 需要从昇腾社区下载安装，在无 NPU 环境下无法完全运行

来源: README.md - 快速上手章节
