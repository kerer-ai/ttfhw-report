遇到的问题 (5)

首次构建时bdist_wheel失败，提示invalid command 'bdist_wheel'

方案: 安装wheel包后重试成功

来源: 构建日志

容器DNS解析失败（getaddrinfo() thread failed to start）

方案: 使用--network host模式并配置--dns 223.5.5.5解决

来源: 容器启动日志

安装CUDA版PyTorch导致UT链接失败（缺少CUDA运行时库）

方案: 重新安装CPU版PyTorch（--extra-index-url https://download.pytorch.org/whl/cpu）

来源: 构建日志

torchair导入失败：libge_local_stub.so undefined symbol

方案: 该问题无法在CPU容器中解决，需要昇腾SDK运行时环境（CANN）

来源: 运行日志

UT构建链接失败：缺少昇腾SDK运行时库符号

方案: 该问题无法在CPU容器中解决，需要安装CANN SDK

来源: 构建日志
