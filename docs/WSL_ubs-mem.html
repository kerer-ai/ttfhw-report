方案: 使用 --privileged --network host 模式启动容器

方案: 从宿主机 /root/ttfhw-work/ubs-comm-20260616165745/dist/ 复制 hcom 头文件和库到容器

来源: ubs-comm-lib (hcom) 非 openEuler 标准仓库包

测试链接失败：/usr/bin/ld: cannot find -lasan

方案: dnf install -y libasan
