遇到的问题 (4)

CMake 配置阶段未找到 Python3 Development.Module 组件

方案: 安装 python3-devel 包: dnf install -y python3-devel

来源: CMake FindPython3 模块需要 Python3 开发头文件和库

Python UT 286 个测试失败，错误为 RuntimeError: Compiler executable is not found

方案: 需要安装 CANN toolkit 以获取 bisheng 编译器和 ld.lld 链接器，设置 PYASC_COMPILER 和 PYASC_LINKER 环境变量

来源: python/asc/runtime/compiler.py 第106行: shutil.which(os.environ.get('PYASC_COMPILER', 'bisheng'))

C++ Lit 测试运行时找不到 llvm-lit

方案: pip install lit 并创建符号链接: ln -sf /usr/local/bin/lit /tmp/llvm-19.1.7-x86_64/bin/llvm-lit

来源: LLVM 预编译包不包含 llvm-lit，需单独安装

样例运行失败，缺少 CANN 仿真器运行库

方案: 需要安装 CANN toolkit 并设置 LD_LIBRARY_PATH 和 ASCEND_HOME_PATH 环境变量

来源: python/asc/runtime/config.py 第76行: 检查运行时库可用性
