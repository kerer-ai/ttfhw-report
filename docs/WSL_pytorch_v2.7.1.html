遇到的问题 (3)

CMake 4.0.0 与项目 cmake_minimum_required(VERSION 3.18) 不兼容

方案: 设置环境变量 CMAKE_POLICY_VERSION_MINIMUM=3.5，不修改源码

来源: ci/build.sh

torch_npu 导入失败，缺少 libhccl.so（CANN 库）

方案: 此为硬件依赖问题，需要 Ascend NPU 硬件和 CANN 软件环境才能运行，非构建问题

来源: test_autocast.py

pip 安装测试依赖超时（网络问题）

方案: 手动安装关键依赖 expecttest 和 hypothesis

来源: pip install
