遇到的问题 (4)

dnf builddep 失败：libisula-devel > 2.1.5-0 不在 openEuler 24.03 仓库中

方案: 从 LCR 源码仓库 (openeuler/lcr) 构建 libisula 2.1.5，安装后 cmake 检查通过

libisula 2.1.4 与 iSulad master (2.1.6) API 不兼容：缺少 isula_sub_string、isula_fopen 及 shim_client_process_state 新字段

方案: 构建并安装 libisula 2.1.5，提供所需 API

cmake -DENABLE_UT=ON 失败：include(Dart) 找不到 Dart 模块

方案: cmake 3.27.9 中 cmake_minimum_required(VERSION 2.8...4.0) 将 CMP0145 设为 NEW，移除了 Dart 模块。通过创建 stub Dart.cmake 并设置 CMAKE_MODULE_PATH 绕过

cgroup_cpu_ut 测试失败 (test_common_find_cgroup_mnt_and_root)

方案: 容器中 cgroup 挂载点与物理机不同，预期 cgroup 路径检查失败，属于环境差异而非代码缺陷
