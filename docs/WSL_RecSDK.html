遇到的问题 (5)

pylint 和 bandit pre-commit hooks 因宿主机 Python 3.9.9 不满足 >=3.10 要求而无法运行

方案: 跳过这两个 hooks，在报告中记录

来源: .pre-commit-config.yaml 中 pylint v4.0.5 和 bandit 1.9.4 要求 Python >=3.10

fbgemm_gpu==1.1.0+cpu 在 aarch64 架构上无可用版本

方案: 无法解决，这是 PyTorch 官方源的限制。fbgemm_gpu 的 CPU wheel 仅提供 x86_64 架构

来源: pip install fbgemm_gpu==1.1.0+cpu --index-url https://download.pytorch.org/whl/cpu

torchrec 安装失败，依赖 fbgemm_gpu

方案: 无法解决，torchrec 强依赖 fbgemm_gpu，而 fbgemm_gpu 在 aarch64 上不可用

来源: pip install torchrec

Python 测试 test_ok 失败，缺少 tensorflow 模块

方案: tensorflow 的 NPU 适配版需从华为 Ascend 仓库下载，不在 PyPI 上。且 TF 组件要求 Python 3.7.5，与容器中的 Python 3.11.6 不兼容

来源: validator/test_safe_checker.py::TestFileSafeCheck::test_ok

C++ 组件构建失败，缺少 Ascend CANN SDK 和 opensource 目录

方案: CANN SDK 是华为昇腾专有软件，需从 https://www.hiascend.com/cann/download 下载安装。opensource 目录需手动创建并放入 pybind11、securec 等依赖

来源: training/common/src/CMakeLists.txt 和 build.sh
