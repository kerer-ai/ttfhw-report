来源: Environment adaptation

WSL file system clock skew warning during build

方案: Non-fatal warning, build completed successfully. This is a known WSL issue with Windows file system time synchronization.

来源: WSL platform limitation

Linux capabilities tests fail in WSL environment

方案: Expected behavior - WSL does not implement full Linux capabilities subsystem. Tests would pass on native openEuler Linux.

来源: WSL platform limitation

Log rotation tests fail in WSL environment

方案: Likely WSL file system timestamp/permission issue. Tests would pass on native openEuler Linux.

来源: WSL platform limitation

来源: Hardware/software dependency
