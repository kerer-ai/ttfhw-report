遇到的问题 (5)

项目默认需要bisheng编译器，但该编译器未在PATH中找到

方案: 通过安装CANN toolkit获得bisheng编译器

来源: CMakeLists.txt:14-20

需要ASCEND_HOME_PATH环境变量（CANN安装路径）

方案: 下载和安装CANN toolkit，并设置正确的环境变量

来源: CMakeLists.txt:56-60

单元测试构建失败，bisheng编译器在编译device侧代码时出现段错误

方案: 需要在有昇腾硬件的环境中进行完整测试

来源: tests/unittest build output

单元测试运行失败，aclshmem_unittest可执行文件不存在

方案: 需要先成功构建单元测试

来源: scripts/run.sh output

示例程序运行失败，缺少Python依赖、CANN工具和运行时库

方案: 需要安装ml_dtypes/torch/scipy，并在有昇腾硬件的环境中运行

来源: scripts/run_examples.sh output
