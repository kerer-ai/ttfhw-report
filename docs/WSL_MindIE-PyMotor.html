遇到的问题 (7)

Docker volume mount 在 Windows Docker Desktop 中失败

方案: 改用 docker cp 复制文件到容器内

dnf install gcc/gcc-c++ 因 openEuler 镜像源下载速度极慢（~10KB/s）多次超时

方案: 项目实际不需要 gcc（纯 Python 项目，依赖均有预编译 wheel），跳过 gcc 安装

容器中无 pip，仅有 python3

方案: 使用 python3 -m ensurepip --upgrade 安装 pip

build.sh 和 generate_proto.sh 使用 'python' 命令，但 openEuler 仅有 'python3'

方案: 创建符号链接 ln -s /usr/bin/python3 /usr/bin/python

脚本文件有 CRLF 行尾（Windows git checkout 导致）

方案: 使用 sed -i 's/\r$//' 转换 LF

容器中无 find 命令（findutils 未安装）

方案: 从 OS 仓库 dnf install findutils（小包，下载较快）

tests/run_tests.sh 中 pip install pytest-xdist 失败（pip 命令不存在，仅有 pip3）

方案: pytest-xdist 已通过 requirements 预装，脚本中缺失不影响测试运行
