遇到的问题 (6)

Docker pull failure from registry-1.docker.io

方案: Used mirror docker.1ms.run/ubuntu:24.04 and tagged as ubuntu:24.04

来源: Network workaround

Python 3.11 not available in Ubuntu 24.04 default repositories

方案: Added deadsnakes PPA: add-apt-repository ppa:deadsnakes/ppa

来源: Standard Ubuntu practice

Shell script line endings (\r\n) causing execution failures

方案: Ran find /workspace -name '*.sh' -exec dos2unix {} \;

来源: Standard Windows-to-Linux issue

msServiceProfiler/Tracer.h header not found during ms_coordinator compilation

方案: Cloned https://gitcode.com/Ascend/msit.git and copied msserviceprofiler/include/msServiceProfiler/ to /usr/local/Ascend/ascend-toolkit/latest/include/

来源: User instruction

Tracer.h still not found after copying - ASCEND_HOME_PATH not set

方案: Set ASCEND_HOME_PATH=/usr/local/Ascend/ascend-toolkit/latest before build. The CMake uses $ENV{ASCEND_HOME_PATH}/include for the header search path.

来源: CMakeLists.txt analysis: coordinator CMakeLists.txt line 55

Unit test build fails with source code syntax errors

方案: Not resolved - pre-existing source code bugs in DynamicConfigHandler.cpp (corrupted catch block) and AlarmConfig.cpp (mismatched braces). These errors only appear with -DUT_FLAG=ON compilation flag.

来源: Source code issue - not environment related
