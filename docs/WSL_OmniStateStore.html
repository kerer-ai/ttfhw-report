遇到的问题 (4)

git命令在容器中不可用

方案: 设置 CI_BUILD=1 环境变量跳过submodule更新, 在宿主机手动初始化子模块后挂载到容器

来源: build.sh 第200-203行

release构建失败: mvn (Maven) command not found

方案: 使用 --ut 参数构建 (bash scripts/build.sh -t debug --ut), 该路径调用 make build_cpp 跳过Java插件构建。Java插件构建不在UT测试范围内。

来源: CMakeLists.txt build_all target + build.sh BUILD_UT判断逻辑

文档指定架构为 aarch64 (鲲鹏920), 但实际在 x86_64 上编译

方案: 编译成功, 代码在x86_64架构上可正常编译 (SVE特性仅在BUILD_SVE=ON且aarch64时启用)

来源: CMakeLists.txt BUILD_SVE条件编译
