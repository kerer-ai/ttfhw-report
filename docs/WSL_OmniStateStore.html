遇到的问题 (3)

构建时 Java JAR 打包失败: mvn: command not found

方案: 安装 Maven (dnf install -y maven)，Maven 是 CMakeLists.txt build_all 目标中 Java 构建的必要工具

来源: CMakeLists.txt 第 153-165 行 build_all 目标，调用 make build_bss_java_version_* 时需要 Maven

UT 二进制链接失败: /usr/bin/ld: cannot find -laio

方案: 安装 libaio-devel (dnf install -y libaio-devel)

来源: test/llt/CMakeLists.txt 中 target_link_libraries 指定了 aio 库

文档指定的测试执行工具 hdt 不可公开获取

方案: 采用备选方案: 通过 CMake 直接构建 UT 二进制 (bash scripts/build.sh -t debug --ut)，然后直接运行 ./build/test/llt/bss_ut

来源: test/run_dt.sh 依赖 hdt 命令
