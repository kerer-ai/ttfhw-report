遇到的问题 (4)

CANN Toolkit (1.37 GB) 下载耗时约 16 分钟

方案: 使用 wget 后台下载，未做特殊处理

来源: -

CANN Toolkit 首次安装失败: ge-compiler 子包安装失败

方案: 根因为 pip3 未安装，导致 ge-compiler 安装过程中需要 pip install llm_datadist 失败。安装 python3-pip 后重新安装 CANN Toolkit 成功

来源: /var/log/ascend_seclog/ascend_toolkit_install.log - ERROR: pip3 is not installed

Python 3.10 在 Ubuntu 24.04 中不可用（推荐版本 <= 3.10）

方案: 使用 Ubuntu 24.04 默认的 Python 3.12.3，编译和测试均正常运行

来源: -

示例程序无法执行（需要 NPU 硬件或仿真器）

方案: 仅进行编译验证（编译态），确认编译成功。实际运行需要 Ascend NPU 设备或配置 cannsim 仿真环境

来源: docs/quick_start.md - 编译态 vs 运行态说明
