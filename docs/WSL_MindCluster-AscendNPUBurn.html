遇到的问题 (5)

pre-commit hooks 报告大量代码规范问题（trailing-whitespace, ruff-check, ruff-format, codespell, pylint 等）

方案: 均为代码风格/格式问题，属于正常静态检查反馈，未修改源码（技能约束：不改源码）

来源: pre-commit run --all-files 输出

torch_npu 无法安装（依赖 Ascend NPU 硬件 + CANN 软件包）

方案: 安装 CPU 版 PyTorch (torch-2.12.1+cpu)，C++ 扩展编译自动跳过

来源: build/setup.py 中的 try/except ImportError 逻辑

pytest 不识别 --timeout 参数

方案: 需安装 pytest-timeout 插件：pip3 install pytest-timeout

来源: test/run_test.sh 中使用了 --timeout 参数但未声明依赖

CLI 工具 npu-burn 无法启动（npu_backend.py 导入 torch_npu 失败）

方案: 需在 Ascend NPU 环境中运行，或修改代码使 torch_npu 为可选依赖

来源: ascend_npu_burn/common/npu_backend.py:16 import torch_npu

openEuler 基础镜像不包含 pip3，需先通过 dnf 安装 python3-pip

方案: 调整脚本顺序：先执行 dnf install 安装系统工具链（含 python3-pip），再执行 pip install

来源: openEuler 24.03 LTS 基础镜像
