bash build.sh 3rd 编译 zlib 时 tar 无法解压 .xz 源码包（xz: Cannot exec）

方案: dnf install -y xz 安装后重试，成功

方案: grpc 脚本将自编译 OpenSSL 3.0.9（无 EVP_md2）注入 LD_LIBRARY_PATH，破坏系统 cmake；通过 LD_PRELOAD=/lib64/libcrypto.so.3.0.12（含 EVP_md2，为超集）使构建工具正常加载，重试成功

来源: build.sh 调用 git 获取版本信息 + scripts 依赖 numpy
