summary
The libipmi repository cannot be fully built in a plain Ubuntu 24.04 Docker container using only apt packages and meson. ~370/375 targets build successfully (all libmcpp subproject libraries), but 5 targets fail due to conan-provided shared memory API members.
root_cause
The top-level libipmi code (filter.h) has a direct dependency on conan-provided dbus/shm_tree/shared_memory.h types that are not available in the pure meson build path.
