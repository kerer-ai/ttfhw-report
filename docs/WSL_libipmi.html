遇到的问题 (5)

openEuler 上 python3-ninja 与 ninja-build 包冲突，meson 安装失败

方案: 分步安装依赖，使用 dnf install -y meson --allowerasing 解决冲突

来源: openEuler 24.03 包管理系统

meson setup 失败：缺少 git 程序，无法克隆 libmcpp 子项目

方案: 安装 git (dnf install -y git)

来源: meson.build 第101行 subproject fallback 机制需要 git

SSH 认证失败：libmcpp.wrap 使用 git@gitcode.com SSH URL，容器内无 SSH 密钥

方案: 修改 subprojects/libmcpp.wrap 将 SSH URL 改为 HTTPS URL

来源: subprojects/libmcpp.wrap

meson setup 失败：libmcpp 子项目 mcdatabase 组件找不到 sqlite3 依赖

方案: 安装 sqlite-devel (dnf install -y sqlite-devel)

来源: libmcpp 子项目 mcdatabase/meson.build 第78行

编译失败：1) 缺少 lauxlib.h (LuaJIT 头文件)；2) sqlite3_preupdate_* 符号未定义（系统 SQLite 不含 session extension）

方案: 1) 安装 luajit-devel；2) 从源码编译 SQLite 3.42.0 并启用 -DSQLITE_ENABLE_SESSION -DSQLITE_ENABLE_PREUPDATE_HOOK，安装到 /usr/local

来源: libmcpp 子项目 mcapp/src/expression/lua_expression_evaluator.cpp 第34行 和 mcdatabase/src/sqlite_wrapper.cpp 第423行
