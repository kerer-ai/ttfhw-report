CANN 环境脚本未找到 (set_env.sh)

方案: 挂载宿主 /usr/local/Ascend 目录到容器内

来源: Dockerfile 环境变量 ASCEND_TOOLKIT_HOME

securec.h 头文件未找到

方案: 挂载完整 CANN driver 目录，确保头文件路径完整

来源: CMakeLists.txt 中 include_directories

AudioUtils.cpp 编译错误：std::function 未声明

方案: 添加 #include <functional> 到源文件

来源: AccSDK/source/utils/AudioUtils.cpp

SWIG 生成代码编译错误：PyUnicode_AsUTF8 等 Python C API 未声明

方案: 注释掉 CMakeLists.txt 中的 -DPy_LIMITED_API=0x03090000

来源: AccSDK/source/py/CMakeLists.txt

libbz2.so.1.0 共享库缺失

方案: 创建符号链接 ln -sf /usr/lib64/libbz2.so.1 /usr/lib64/libbz2.so.1.0

来源: openEuler 24.03 中 bzip2 库版本命名差异

cv2 模块未找到 (ModuleNotFoundError)

方案: pip3 install opencv-python-headless

来源: mm.core.frame_selector 运行时依赖

gtest 子模块未初始化，googletest 目录为空

方案: git clone https://github.com/google/googletest.git 到对应目录

来源: 仓库 .gitmodules 未在容器内正确初始化

gtest 链接错误：自编译 gtest 1.16.0 头文件与系统 gtest 1.14.0 库 ABI 不兼容

方案: dnf remove gtest-devel gtest，仅使用自编译版本

来源: 测试 CMakeLists.txt 中 target_link_libraries(... gtest)
