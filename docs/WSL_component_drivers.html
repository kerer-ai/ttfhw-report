遇到的问题 (8)

Docker 在 docker-desktop WSL distro 中不可用

方案: 在 Ubuntu WSL distro 中安装 docker.io CLI 客户端，通过 Docker Desktop 的 WSL2 集成使用

来源: Docker Desktop WSL2 集成文档

容器意外停止

方案: 使用 docker start 重新启动容器

来源: 多次发生，可能因资源限制导致

libmcpp.wrap 使用 SSH URL (git@gitcode.com:openUBMC/libmcpp.git)，meson 无法克隆

方案: 手动通过 HTTPS 克隆到 subprojects/libmcpp/

来源: subprojects/libmcpp.wrap 第2行

openEuler 的 sqlite3 不包含 SQLITE_ENABLE_PREUPDATE_HOOK 符号（sqlite3_preupdate_hook 等）

方案: 从 openEuler 源码 RPM 下载 sqlite3 源码，添加 CFLAGS=-DSQLITE_ENABLE_PREUPDATE_HOOK 重新编译安装

来源: subprojects/libmcpp/libraries/mcdatabase/meson.build:19

openEuler 的 boost-devel 不提供 pkg-config (.pc) 文件

方案: 手动创建 /usr/lib64/pkgconfig/boost.pc 和 boost_program_options.pc

来源: drivers/meson.build 使用 dependency('boost', method: 'pkg-config')

LuaJIT 不提供 lua_pcallk 等 Lua 5.2+ API

方案: 创建 luajit.pc 指向系统 lua 5.4 库（-llua），使 dependency('luajit') 解析到正确库

来源: subprojects/libmcpp/libraries/mcapp/meson.build:97

源码 bug: power.h 使用 OpticalModule_Oem* 但缺少前向声明（最近提交 !580 引入的回归）

方案: 在 power.h 的 namespace dev 中添加 class OpticalModule_Oem; 前向声明

来源: drivers/pcie_nic_card/wangxun/interface_pldm/optical_module/power.h:42
