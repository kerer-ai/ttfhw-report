方案: pip3 install torch==2.7.1 --index-url https://download.pytorch.org/whl/cpu

方案: 删除build/pytorch/build目录后重新构建

来源: CMake错误信息提示

来源: docs/zh/INSTALL.md - GCC版本兼容性表格明确规定PyTorch v2.7.1需要GCC 11.2

CANN (Ascend NPU SDK) 依赖无法满足

方案: 未解决。CANN是华为昇腾专有软件，需从昇腾社区(hiascend.com)下载，无法通过标准包管理器安装。即使编译成功，链接阶段仍需要CANN库(libascend_hal.so等)

来源: docs/zh/INSTALL.md - 配置CANN环境变量章节
