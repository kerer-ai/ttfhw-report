方案: git config --global --add safe.directory '*'（仓库从 WSL 宿主机挂载进容器，git 安全检查所致，属环境配置）

来源: git 错误提示

方案: dnf install -y python3-devel（tests/CMakeLists.txt:25 find_package(Python3 COMPONENTS Interpreter Development REQUIRED) 需要 Python 开发头文件）

来源: tests/CMakeLists.txt:25

requirements.txt 中 torch_npu==2.7.1.post4 与 CPU 版 torch 安装冲突，无 NPU 环境无法安装 torch_npu

方案: 安装 torch==2.7.1+cpu 替代 CUDA 版 torch，并剔除 torch_npu（tests/conftest.py 已用 MagicMock 注入 torch_npu，UT 无需真实 torch_npu 包）
