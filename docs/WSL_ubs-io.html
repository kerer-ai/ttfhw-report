遇到的问题 (6)

Windows 换行符导致 shell 脚本执行失败

方案: 使用 dos2unix 命令转换脚本格式

来源: 通用 Windows/Linux 兼容性问题

缺少 git 命令导致 CMake FetchContent 失败

方案: 安装 git 包: dnf install -y git

来源: CMake FetchContent 依赖 git

缺少 infiniband/verbs.h 头文件

方案: 安装 libibverbs-devel (rdma-core-devel) 包

来源: ubs-comm 依赖 RDMA 库

缺少 patch 命令导致 mockcpp 补丁应用失败

方案: 安装 patch 包: dnf install -y patch

来源: mockcpp 构建需要打补丁

openEuler 默认仓库 SSL 连接失败

方案: 配置清华镜像源: mirrors.tuna.tsinghua.edu.cn

来源: 网络环境问题

lcov 未在 openEuler 仓库中找到

方案: 跳过覆盖率报告，直接运行测试二进制文件

来源: lcov 不在 openEuler 24.03 标准仓库中
