遇到的问题 (4)

log/CMakeLists.txt文件缺失导致cmake配置失败

方案: 从git历史恢复：git checkout 86be28a -- 3rdparty/log/CMakeLists.txt

来源: 文件存在于项目git历史中

方案: 更新memfabric_hybrid到最新：git checkout origin/master

来源: 项目代码使用了最新版本的结构体成员

UT构建缺少patch工具

方案: dnf install patch

来源: mockcpp子模块需要patch处理

某些UT测试需要NPU环境(libubsio_kvc.so)

方案: 跳过需要NPU环境的测试，运行基础测试

来源: test_mmc_test等测试依赖NPU库
