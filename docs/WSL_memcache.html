GitHub 不可访问，git submodule update 失败

方案: 使用 git config url.insteadOf 将 github.com 重定向到 gitcode.com/gh_mirrors 和 gitclone.com

来源: .gitmodules 中 spdlog, cpp-httplib, nlohmann/json, googletest, mockcpp, prometheus-cpp-lite 使用 GitHub URL

来源: gitclone.com shallow clone 限制

prometheus-cpp-lite v2 目录结构变化: simpleapi/ 和 core/ 独立目录改为统一 include/ 目录

方案: 从 ghproxy.net 下载正确版本 tarball (commit 48d09c4) 替换

来源: .gitmodules 指定 branch=master 无版本锁定

spdlogger4c.h:55 int32_t 未声明，spdlog v1.15.3 不再传递包含 <cstdint>

方案: 设置环境变量 export CXXFLAGS="-include cstdint" 强制包含
