遇到的问题 (3)

容器内 pre-commit 报 getaddrinfo() thread failed to start，无法 git fetch hooks 仓库

方案: 为容器添加 --security-opt seccomp=unconfined（宿主机 CentOS 7 内核 4.18 与 openEuler 24.03 容器 glibc 多线程 DNS 解析在默认 seccomp 下不兼容）

来源: 环境兼容问题

import mindspeed_llm 报 No module named 'torch_npu'

方案: 记录为硬件/依赖限制，不修改源码、不强行 mock；本机无 NPU 设备，cann 基础镜像未预装 torch_npu wheel

来源: 硬件依赖限制

pre-commit 中部分 hook 自动修改源码

方案: 执行结束后立刻 git checkout -- . && git clean -fdx 完整还原，符合源码不可修改原则

来源: 源码保护措施
