遇到的问题 (4)

初次构建失败：Could not find ASCConfig.cmake

方案: 挂载宿主机已安装的 CANN 目录到容器

来源: cmake/dependencies.cmake 显示需要 CANN 软件栈

运行时库缺失：libmetadef.so 无法加载

方案: 设置 LD_LIBRARY_PATH=/usr/local/Ascend/cann-9.1.0/lib64

来源: CANN 安装目录结构

asc_opc 编译器找不到

方案: source /usr/local/Ascend/cann-9.1.0/set_env.sh 配置完整环境

来源: CANN set_env.sh 脚本设置 PATH 和其他环境变量

ops-tensor git submodule checkout 失败

方案: git submodule init && git submodule update

来源: ops-tensor submodule 需要初始化
