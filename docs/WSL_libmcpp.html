遇到的问题 (5)

openEuler 仓库 python3-ninja 与 ninja-build 包冲突，meson 无法直接安装

方案: 先单独安装 ninja-build，再安装 meson。安装过程自动升级 python3-ninja 并解决依赖

来源: openEuler 24.03 dnf 仓库包冲突

系统 SQLite 3.42.0 未启用 SQLITE_ENABLE_PREUPDATE_HOOK，导致 mcdatabase 链接时 undefined reference 到 6 个 preupdate 符号

方案: 从源码编译 SQLite 3.45.0 安装到 /usr/local，CFLAGS='-DSQLITE_ENABLE_PREUPDATE_HOOK -DSQLITE_ENABLE_UPDATE_DELETE_LIMIT'，通过 PKG_CONFIG_PATH=/usr/local/lib/pkgconfig 让 meson 发现。下载 266s + 编译 17s

来源: libraries/mcdatabase/meson.build 第20行硬编码该编译宏

容器内无 D-Bus daemon，libmcpp 测试套件 113 个 D-Bus 相关测试失败 + 2 个测试套件崩溃

方案: dnf install -y dbus-daemon dbus diffutils。安装后 libmcpp D-Bus 测试套件从全部失败 → 全部通过

来源: 测试固件 TestWithDbusDaemon 需要 fork+exec dbus-daemon 二进制文件

/dev/shm 残留的 SHM FIFO 文件干扰 mcengine 测试

方案: 测试前执行 rm -rf /dev/shm/* 清理共享内存残留
